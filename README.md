# Vue 3 + Vuetify 3 Static UI Project

This project is a simple frontend UI built using **Vue 3** and **Vuetify 3**, deployed on **Netlify**.  
It contains static pages demonstrating layout, styling, and responsive UI development using Vuetify's grid system and component library.
